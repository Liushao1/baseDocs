> #### 17. 角色菜单权限 s_role_menu

作用：角色菜单权限表

| 字段名 | 字段类型 | 长度 | 备注                               |
| ------ | -------- | ---- | ---------------------------------- |
| id     | int      | 11   | 角色菜单权限ID                     |
| roleId | int      | 11   | 归属角色（引用 **s_role** 角色ID） |
| menuId | int      | 11   | 归属菜单（引用 **s_menu** 菜单ID） |
