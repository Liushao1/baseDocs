> #### 5. 表单 s_formlayout

作用：所有表单和实体间的关系 表单包括（添加页面，修改页面，查看页面）

| 字段名      | 字段类型 | 长度 | 备注                                |
| ----------- | -------- | ---- | ----------------------------------- |
| id          | int      | 11   | 表单ID                              |
| name        | varchar  | 255  | 表单名称                            |
| pk_entity_1 | varchar  | 255  | 归属实体 （引用 **s_entity** 表ID） |
| type        | int      | 11   | 表单类型                            |
| coding      | text     | 11   | 页面JS                              |
