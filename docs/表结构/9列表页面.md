> #### 9.列表页面 s_listlayout

作用：系统所有的列表页面

| 字段名          | 字段类型 | 长度 | 备注                                |
| --------------- | -------- | ---- | ----------------------------------- |
| id              | int      | 11   | 列表ID                              |
| name            | varchar  | 255  | 列表名称                            |
| pk_listlayout_1 | int      | 11   | **暂不使用**                        |
| pk_entity_1     | int      | 11   | 归属实体（引用 **s_entity** 表ID ） |
| type            | int      | 11   | 实体默认弹出窗口使用列表            |
